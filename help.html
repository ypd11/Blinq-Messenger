Getting Started

1. Open Blinq Messenger

Run Blinq Messenger on two or more computers connected to the same private Wi-Fi or Ethernet network, or switch to Internet Mode to use a Blinq account with approved remote contacts.

2. Allow the firewall prompt

When Windows Firewall asks, allow Blinq Messenger on Private networks. Local discovery may not work on Guest Wi-Fi or restricted VPN networks.

3. Find contacts

In LAN Mode, contacts appear automatically when they are running Blinq Messenger on the same LAN and are not Invisible. In Internet Mode, add contacts by Blinq ID.

4. Start chatting

Double-click a contact, or right-click and choose Open Chat.

LAN Mode discovers peers locally and sends messages directly between devices whenever possible. Internet Mode uses the Blinq Internet service for account-based remote private chat.

Profile and Status

Your profile panel shows your avatar, display name, personal message, current status, and optional playing media.

  • Use Profile > Change Display Name or the small edit button in the profile panel to rename yourself.
  • Use Profile > Set Avatar to choose your display picture.
  • Edit the personal message directly in the top panel. The default is Hi, let's chat!.
  • Use the status dropdown for Available, Busy, Away, Do Not Disturb, Invisible, or custom status messages.
  • When enabled, Blinq Messenger can show Windows media playback as Playing: ....

Private and Public Chat

Private chat
Double-click a contact, type a message, and press Enter or click Send. Private chat works in LAN Mode and Internet Mode.
Public chat
Use Chat > Public Chat for a LAN-wide room. Public chat is a LAN Mode feature.
Rich messages
Use the rich-message button to style text with font, size, bold, italic, underline, and color.
Emoji
Use the emoji button to insert expressive characters into chat.
Conversation search
Use the Find button or press Ctrl+F inside a private chat to search that conversation.
Whistle
Send a quick attention sound to a contact.
Typing and read status
Contacts can see typing state and message receipts when available.

Files and Images

Use the attachment button in a private chat or right-click a contact and choose Send File.

  • Images that can be displayed in chat are sent directly and shown inline.
  • Other files ask the receiver for acceptance before transfer.
  • Received files are saved in Downloads under Blinq Messenger Received.
  • Be careful with files from other users. Only open files from people and networks you trust.

Drawing Pad

The drawing button in a private chat opens a small drawing pad. Drawings are sent as images and appear directly in the chat.

  • Use color swatches or the Color button to choose ink.
  • Use the size slider to change brush size.
  • Use Pen or Eraser, plus Undo, Redo, and Clear.
  • Click Send Drawing to send the image.

Contacts, Groups, Favorites, and Blocking

The contact list is organized into Favorites, custom groups, Online, Away / Idle, Busy, and Offline.

  • Right-click a contact to open chat, view contact info, send a file, whistle, favorite, move to a group, clear history, or block.
  • Drag contacts between custom groups.
  • Use Chat > Manage Contact Groups... to create, rename, or delete custom groups.
  • Blocked users are hidden and their messages/files/public chat posts are ignored.

Internet Mode

Internet Mode lets you sign in with a Blinq ID and chat privately with approved contacts through the Blinq Internet service.

  • Use Profile > Switch to Internet Mode to sign in or create a Blinq account.
  • Use Chat > Add Contact... to search by display name or Blinq ID, or to add someone directly when you already know their full Blinq ID.
  • Internet contacts can exchange private messages, images, typing state, read receipts, whistles, profile details, and presence when the server is reachable.
  • Use Profile > Settings and turn off Allow people to find me in Internet search if you only want people to add you when you share your Blinq ID yourself.
  • LAN-only tools such as Public Chat, Direct Connect by IP, LAN groups, subnet scan, and firewall helpers are available in LAN Mode.
  • If the Internet server is unavailable, Blinq Messenger can return to LAN Mode so local chat remains usable.

Themes

Use the Theme menu to change the app globally. Open chat windows update live, and contacts can share their theme color so their avatar frame appears in their chosen color.

Current theme choices include Sky, Aqua, Mint, Rose, Sunset, Violet, Orchid, Ruby, Midnight, and Graphite.

Settings and Tray

  • Open Profile > Settings to adjust behavior, sounds, notifications, history, startup, and blocked users.
  • Enable Hide my typing indicator if you do not want contacts to see when you are typing.
  • Disable Allow people to find me in Internet search to opt out of Internet Mode account discovery.
  • Enable Send an away auto-reply to reply once every few minutes per contact while your status is Away, Busy, Idle, or a custom status.
  • Blinq Messenger can keep running in the system tray near the clock.

Updates and Backups

  • Use Help > Check for Updates to check the official update file, read the changelog, and download a newer installer when one is available.
  • After downloading an update, use Run Installer to close Blinq Messenger and start the installer automatically.
  • Use Profile > Settings > Reset > Backup Data to export your settings and chat history to a .blinqbackup file.
  • Use Profile > Settings > Reset > Restore Data to import a backup file. Restoring replaces current settings and chat history, then restarts the app automatically.
  • The installer is designed to keep user data, but keeping a backup before major updates is recommended.

Network and Ports

In LAN Mode, Blinq Messenger uses a local UDP discovery channel to find nearby users, then direct TCP connections for private messages, file transfers, drawings, and direct-connect handshakes. In Internet Mode, private chat uses the configured Blinq Internet server.

  • Use Chat > Network Diagnostics to see the actual LAN discovery state and the TCP port this copy of Blinq Messenger is using right now.
  • Use Chat > Copy Connection Invite to copy your current IP and TCP port for someone else to paste into Direct Connect.
  • Use Chat > Reconnect Saved Contacts to try remembered contacts and saved manual IPs again.
  • Use Chat > Scan Local Network as an optional fallback when discovery is blocked. It checks nearby local addresses using Direct Connect.
  • Use Chat > Firewall Setup Helper for Windows Firewall guidance.
  • If the preferred direct TCP range is busy, Blinq Messenger can listen on an OS-assigned fallback TCP port and announce that port to discovered peers.
  • If UDP discovery cannot bind or broadcast traffic is blocked, automatic contact discovery may be incomplete.
  • Use Chat > Direct Connect by IP in LAN Mode when you know another computer's IPv4 address. If that user is on a fallback TCP port, enter the exact address as IP:port.
Port details should come from Network Diagnostics because firewalls, other running apps, and startup order can change what the app is actually using.

FAQ

Why do I not see another user?

Make sure both computers are on the same private network, Windows Firewall allows Blinq Messenger on Private networks, and neither user is set to Invisible.

Does Blinq Messenger use a server?

LAN Mode does not require a central chat server for local discovery and peer-to-peer communication. Internet Mode uses the Blinq Internet service for account-based remote private chat.

Can I chat with someone outside my LAN?

Yes. Switch to Internet Mode, sign in with a Blinq ID, and add the other person as a contact. Direct Connect by IP is still a LAN Mode fallback for reachable local networks when broadcast discovery is blocked.

What happens if a port is already in use?

For direct TCP chat and file transfer, Blinq Messenger tries its preferred range and can fall back to another available TCP port. Peers discovered automatically receive the active port in your presence message. If discovery is blocked, Direct Connect may need the exact IP:port shown in Network Diagnostics.

What if discovery works for one user but not another?

The user whose discovery channel is unavailable may not see contacts automatically. Other users might still see them if outbound announcements are allowed, and direct chats can still work when the active TCP port is reachable.

Why does Blinq remember contacts?

Remembered contacts let Blinq try the last known IP and port again later. This helps when automatic discovery is blocked but the devices are still reachable on the same network.

What does Do Not Disturb block?

When your status is Do Not Disturb, Blinq Messenger refuses incoming private messages, files, drawings, whistles, typing indicators, and activities. If you start a private conversation while in Do Not Disturb, Blinq changes you back to Available first so the other person can reply. Public Chat remains visible.

Why did a file ask for acceptance but an image did not?

Displayable images are shown inline and sent directly. Other files ask for acceptance first so the receiver can decide whether to download them.

Where are received files saved?

Received files are saved in your Downloads folder under Blinq Messenger Received.

What does Invisible do?

In LAN Mode, Invisible stops announcing you on the LAN. Contacts that already know your address may still be able to message you directly. In Internet Mode, presence is sent through your account connection.

Why does my status become Idle?

If you are inactive for a while, Blinq Messenger can show you as Idle. Returning to the app updates activity again.

Is my chat moderated or stored by Exe Innovate?

Blinq Messenger does not moderate your conversations. LAN Mode chats are local/direct. Internet Mode depends on the Blinq Internet service for account-based delivery, and users remain responsible for what they send and receive.
